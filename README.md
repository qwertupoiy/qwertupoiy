# 안녕하세요 유도현입니다!

데이터 흐름을 설계하고 안정적으로 운영하는 백엔드 개발자입니다.  
Spring Boot와 FastAPI를 기반으로 데이터를 수집, 저장, 분석, 시각화하는 백엔드 파이프라인을 직접 구축해왔습니다.  
현재는 백엔드 역량을 기반으로 데이터 엔지니어로 성장하는 것을 목표로 하고 있습니다.
